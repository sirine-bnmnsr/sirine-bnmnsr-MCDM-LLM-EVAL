# Data

Place the following files in this directory before running the notebooks.

| File | Description | Generated by |
|------|-------------|--------------|
| `ollama_mmlu_econometrics_results.csv` | Raw model responses for all 140 MMLU Economics questions across 9 models. Columns: `model, question_id, question, A, B, C, D, correct_answer, model_answer, is_correct, causal_reasoning, raw_output`. | `01_LLM_Querying_and_Question_Selection.ipynb` |
| `LikertScale.csv` | Human evaluation ratings. Columns: `Model, question_number` + one column per sub-criterion (S1.1–S4.2), three rows per (model, question) pair. | Collected via survey |
| `adjusted_evaluator_scores.csv` | Outlier-adjusted version of `LikertScale.csv`. | `03_Aggregation_and_Graphs.ipynb` |
| `dispersion_by_question.csv` | Semantic and structural dispersion metrics per question. Columns: `question_id, dispersion_steps, dispersion_full, ...` | `04_Semantic_Dispersion.ipynb` |

## Data availability

The anonymised `LikertScale.csv` file is available on request from the author.
Raw model outputs (`ollama_mmlu_econometrics_results.csv`) can be regenerated
by running `01_LLM_Querying_and_Question_Selection.ipynb` with Ollama installed
and the nine models pulled (see README.md for instructions).
